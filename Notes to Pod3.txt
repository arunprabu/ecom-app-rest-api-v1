  store the cart items in a variable. 
    let cartItems = [

        "productId": 76,

        "quantity": 2,
        "price": 2232,

      }
    ]

  On Continue button click, 
     https://ecom-app-rest-api-v1.herokuapp.com/orders/1  -- PUT with id 1 and req body  
    req body should be cartItems array 
      xhttp.open("PATCH", "https://ecom-app-rest-api-v1.herokuapp.com/orders/1" ) ;
      xhttp.send( { orderItems: cartItems });

4. Complete Screen
---
  Onload https://ecom-app-rest-api-v1.herokuapp.com/orders/1 - GET
    keep the order id in a variable. 
    display total items and total price in UI.

    and then, display shipping address, billing address, payment info from the above req. 

    when you click complete from this screen

    https://ecom-app-rest-api-v1.herokuapp.com/orders/1  -- PUT with id 1 and req body  
    req body should be {isOrderCompleted: true} being req body
      xhttp.open("PATCH", "https://ecom-app-rest-api-v1.herokuapp.com/orders/1" ) ;
      xhttp.send( { isOrderCompleted: true });

    That's all. 
    upon successful order placement show the confirmation screen with msg.
